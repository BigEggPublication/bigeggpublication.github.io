## 关于作者

**阿普丽尔·斯佩特**（**April Speight**）是一名Python开发人员，并且她热衷于帮助编程的初学者们入门。她相信通过提供易学易懂、相关、并且有趣的编程指令，就能够真正地激发出学习的热情。作为一名真正的创造者，April喜欢编写AI助手以及聊天机器人，创建包含混合现实的体验，以及探索如何向非技术受众传授技术概念的新方法。如果你想对她目前正在学习或创建的东西了解更多，请在Twitter上关注她[@VogueandCode](https://twitter.com/vogueandcode)。

## 关于技术编辑

## 致谢
